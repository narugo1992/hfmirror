Installation
===================

``hfmirror`` is currently hosted on PyPI. It required python >= 3.7.

You can simply install hfmirror from PyPI with the following command:

.. code:: shell

    pip install hfmirror

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/narugo1992/hfmirror.git@main

of ``hfmirror`` should be shown.
